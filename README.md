# SayBetter / 好好说话

一个开源的"高情商回复助手"：帮你把想说的话，改成更体面、更清楚、更不伤人的表达。

## 为什么做这个？

人人都有这样的场景：

- 不知道怎么拒绝别人
- 不知道怎么催人
- 不知道怎么跟领导沟通
- 不知道怎么回相亲对象
- 不知道怎么跟父母解释
- 不知道怎么道歉
- 不知道怎么表达边界
- 不知道怎么把话说得坚定但不攻击

它的爆点在于：结果天然适合截图传播。

## 示例

**输入：**
> 我不想借钱给他，但又不想撕破脸。

**输出：**

**温和版：**
> 我最近手头也有一些安排，可能没办法帮你周转。希望你能理解，不是针对你，只是我现在确实不太方便。

**坚定版：**
> 这次我没办法借钱给你。我们关系归关系，但钱的事情我还是想保持清楚边界。

**高情商版：**
> 我知道你现在可能挺着急的，但这件事我确实帮不上。要是你愿意，我可以帮你一起想想有没有别的解决办法。

## 分享卡片

生成"原话 vs 更好的说法"分享卡片，适合社交平台传播：

```
原话：你怎么又拖到现在？
换种说法：这个事情我们能不能今天定下来？我这边需要根据结果安排后续动作。
```

## 项目结构

```
saybetter/
├── scenarios/          # 场景库（每个场景一个 YAML，可贡献 PR）
│   ├── rejection.yaml
│   ├── apology.yaml
│   ├── workplace.yaml
│   ├── dating.yaml
│   ├── parents.yaml
│   ├── money.yaml
│   └── conflict.yaml
├── prompts/            # 提示词模板
│   ├── zh-CN/
│   └── en-US/
├── web/                # Web 前端
└── extension/          # 浏览器扩展
```

## 场景格式

每个场景是一个 YAML 文件，方便社区贡献：

```yaml
name: 拒绝请求
description: 帮用户温和但清楚地拒绝别人
inputs:
  - relationship
  - original_message
  - desired_tone
outputs:
  - gentle
  - firm
  - warm
  - not_recommended
```
