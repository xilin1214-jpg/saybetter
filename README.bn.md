# SayBetter

> যা বলতে চান, তা বলুন — কাউকে কষ্ট না দিয়ে।

**SayBetter** একটি ওপেন-সোর্স যোগাযোগ সহায়ক যা কঠিন বার্তাগুলোকে স্পষ্ট, সৌম্য এবং কার্যকর অভিব্যক্তিতে পুনরায় লেখে।

## উদাহরণ

```
ইনপুট: তুমি আবার দেরি করছো কেন?

আউটপুট:
  ✅ সৌম্য:    আমরা কি আজ এটা ফাইনাল করতে পারি? আমার পরের পরিকল্পনার জন্য এটা দরকার।
  ✅ দৃঢ়:     এটা আজ ফাইনাল হতে হবে, নাহলে পরের কাজ প্রভাবিত হবে।
              দিনের শেষের আগে একটা স্পষ্ট উত্তর দিন।
  ✅ বুদ্ধিমান: জানি আপনার অনেক কাজ আছে, কিন্তু এই ফলাফল সামনের জন্য গুরুত্বপূর্ণ।
              আজ আমরা কি একটা সিদ্ধান্তে আসতে পারি?
  ❌ এড়িয়ে চলুন: "তুমি সবসময় এরকম" / "আবার সেই একই কথা"
```

## কেন SayBetter?

অনেক মানুষের ভাবনা থাকে — শব্দ থাকে না। SayBetter আপনাকে গুরুত্বপূর্ণ মুহূর্তে স্পষ্টভাবে প্রকাশ করতে সাহায্য করে: প্রত্যাখ্যান, মনে করানো, ক্ষমা চাওয়া, সীমানা নির্ধারণ, তাগিদ দেওয়া।

## বৈশিষ্ট্য

- কঠিন বার্তা ৩-৫টি ভালো সংস্করণে পুনরায় লিখুন
- সম্পর্ক, লক্ষ্য এবং সুর নির্বাচন করুন
- ১২টি পরিস্থিতি-ভিত্তিক প্রম্পট লাইব্রেরি (অবদানের জন্য উন্মুক্ত)
- "অনুচিত" অভিব্যক্তি ও ব্যাখ্যা
- শেয়ারযোগ্য কার্ড (PNG ডাউনলোড)
- ২০টি বাস্তব উদাহরণ
- OpenAI-সম্মত (DeepSeek, Kimi, Ollama ইত্যাদির সাথে কাজ করে)
- ১০টি ভাষা সমর্থিত

| ⏰ মনে করানো | অসম্মানজনক না হয়ে এগিয়ে যাওয়া |

| 💬 অসন্তুষ্টি | ঝগড়া ছাড়া অসন্তুষ্টি প্রকাশ |
| 🚧 সীমানা | আপনার সীমানা স্পষ্টভাবে প্রকাশ |
| 👔 ম্যানেজারের সাথে কথা | পেশাদার ঊর্ধ্বমুখী যোগাযোগ |
| 🤝 সহকর্মীদের সাথে কথা | বন্ধুত্বপূর্ণ এবং দক্ষ যোগাযোগ |
| 💕 সম্পর্ক যোগাযোগ | রোমান্টিক সম্পর্কে অনুভূতি ও চাহিদা |
| 🏠 বাবা-মায়ের সাথে কথা | কঠিন বিষয় ব্যাখ্যা |
| 💰 টাকা | অর্থ সম্পর্কিত অস্বস্তিকর কথোপকথন |
| 📞 অভিযোগ | ফলপ্রসূ অভিযোগ |
| 📋 বেতন ও পদত্যাগ | ক্যারিয়ারের গুরুত্বপূর্ণ মুহূর্তের যোগাযোগ |

## দ্রুত শুরু

```bash
cd saybetter
pnpm install
cp .env.example .env
pnpm dev
```
