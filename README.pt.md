# SayBetter

> Diga o que quer dizer, sem magoar.

**SayBetter** é um assistente de comunicação de código aberto que reescreve mensagens difíceis em expressões claras, gentis e eficazes.

## Exemplo

```
Entrada: Por que você está atrasado de novo?

Saída:
  ✅ Suave:    Podemos finalizar isso hoje? Preciso do resultado para planejar os próximos passos.
  ✅ Firme:    Isso precisa ser finalizado hoje, senão impacta o cronograma.
               Por favor, me dê uma resposta clara até o fim do dia.
  ✅ Empático: Sei que você tem muita coisa, mas esse resultado é crucial para o que vem pela frente.
              Podemos nos alinhar numa conclusão hoje?
  ❌ Evite:   "Você está sempre atrasado" / "Toda vez a mesma coisa."
```

## Por que SayBetter?

Muitas pessoas não têm falta de ideias — têm falta de palavras. SayBetter te ajuda a se expressar com clareza quando importa: recusar, cobrar, pedir desculpas, estabelecer limites, insistir.

## Recursos

- Reescreva mensagens difíceis em 3-5 versões melhores
- Escolha relação, objetivo e tom
- Biblioteca de 12 cenários baseados em prompts (aberta a contribuições)
- Expressões "não recomendadas" com explicações
- Cartões compartilháveis (download PNG)
- 20 exemplos reais
- Compatível com OpenAI (DeepSeek, Kimi, Ollama, etc.)
- 10 idiomas suportados

## Cenários

| Cenário | Descrição |
|---------|-----------|
| ✋ Recusa | Recusar pedidos com calor e clareza |
| ⏰ Cobrança | Avançar sem ofender |
| 🙏 Desculpas | Pedidos de desculpas sinceros |
| 💬 Insatisfação | Expressar frustração sem brigar |
| 🚧 Limites | Expressar seus limites claramente |
| 👔 Falar com seu chefe | Comunicação profissional ascendente |
| 🤝 Falar com colegas | Comunicação amigável e eficiente |
| 💕 Relacionamento | Sentimentos e necessidades no amor |
| 🏠 Falar com seus pais | Explicar coisas difíceis aos pais |
| 💰 Dinheiro | Conversas constrangedoras sobre dinheiro |
| 📞 Reclamações | Reclamações eficazes que trazem resultados |
| 📋 Salário e demissão | Comunicação em momentos-chave da carreira |

## Início rápido

```bash
git clone https://github.com/xilin1214-jpg/saybetter.git
cd saybetter
pnpm install
cp .env.example .env
pnpm dev
```
