# SayBetter

> .قل ما تريد أن تقوله، دون أن تجرح

**SayBetter** هو مساعد تواصل مفتوح المصدر يُعيد صياغة الرسائل الصعبة إلى تعبيرات واضحة ولطيفة وفعّالة.

## مثال

```
الإدخال: لماذا تأخرت مرة أخرى؟

الإخراج:
  ✅ لطيف:    هل يمكننا الانتهاء من هذا اليوم؟ أحتاج النتيجة للتخطيط للخطوات التالية.
  ✅ حازم:    يجب الانتهاء من هذا اليوم، وإلا سيؤثر على الجدول الزمني.
              يرجى إعطائي إجابة واضحة قبل نهاية اليوم.
  ✅ ذكي:     أعرف أن عندك الكثير، لكن هذه النتيجة مهمة لما هو قادم.
              هل يمكننا التوصل لنتيجة اليوم؟
  ❌ تجنّب:  "أنت دائماً متأخر" / "كل مرة نفس الشيء"
```

## لماذا SayBetter؟

الكثير من الناس لا يفتقرون للأفكار — بل يفتقرون للكلمات. يساعدك SayBetter على التعبير بوضوح في اللحظات المهمة: الرفض، المتابعة، الاعتذار، وضع الحدود، والإلحاح.

## المميزات

- أعد صياغة الرسائل الصعبة إلى 3-5 نسخ أفضل
- اختر العلاقة والهدف والنبرة
- مكتبة من 12 سيناريو (مفتوحة للمساهمات)
- تعبيرات "غير مستحسن" مع التفسيرات
- بطاقات قابلة للمشاركة (تحميل PNG)
- 20 مثالاً واقعياً
- متوافق مع OpenAI (DeepSeek، Kimi، Ollama، إلخ)
- يدعم 10 لغات

## السيناريوهات

| السيناريو | الوصف |
|-----------|--------|
| ✋ الرفض | رفض الطلبات بلطف ووضوح |
| ⏰ المتابعة | دفع الأمور قدماً دون إساءة |
| 🙏 الاعتذار | اعتذارات صادقة دون إذلال |
| 💬 الاستياء | التعبير عن الاستياء بدون شجار |
| 🚧 الحدود | التعبير عن حدودك بوضوح |
| 👔 التحدث مع مديرك | تواصل مهني تصاعدي |
| 🤝 التحدث مع الزملاء | تواصل ودود وفعّال |
| 💕 التواصل العاطفي | المشاعر والاحتياجات في العلاقات |
| 🏠 التحدث مع الوالدين | شرح الأمور الصعبة للوالدين |
| 💰 المال | محادثات محرجة عن المال |
| 📞 الشكاوى | شكاوى فعّالة تحقق نتائج |
| 📋 الراتب والاستقالة | التواصل في لحظات مهنية حاسمة |

## البداية السريعة

```bash
git clone https://github.com/xilin1214-jpg/saybetter.git
cd saybetter
pnpm install
cp .env.example .env
pnpm dev
```
