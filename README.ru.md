# SayBetter

> Говорите то, что хотите, не раня других.

**SayBetter** — это открытый помощник по общению, который переписывает сложные сообщения в ясные, доброжелательные и эффективные выражения.

## Пример

```
Ввод: Ты опять опаздываешь?

Вывод:
  ✅ Мягко:   Можем ли мы завершить это сегодня? Мне нужен результат для планирования следующих шагов.
  ✅ Твёрдо:  Это должно быть завершено сегодня, иначе пострадает график.
              Пожалуйста, дайте чёткий ответ до конца дня.
  ✅ Тактично: Знаю, у вас много дел, но этот результат критичен для дальнейшего продвижения.
              Можем ли мы прийти к заключению сегодня?
  ❌ Не надо: "Ты вечно опаздываешь" / "Вечно одно и то же."
```

## Почему SayBetter?

Многим людям не хватает не идей — а слов. SayBetter помогает ясно выражать себя, когда это важно: отказывать, напоминать, извиняться, устанавливать границы, настаивать.

## Возможности

- Перепишите сложные сообщения в 3-5 улучшенных версий
- Выбирайте отношения, цель и тон
- Библиотека из 12 сценариев (открыта для вкладов)
- «Не рекомендуется» с объяснениями
- Карточки для публикации (скачивание PNG)
- 20 реальных примеров
- Совместимость с OpenAI (DeepSeek, Kimi, Ollama и др.)
- Поддержка 10 языков

## Сценарии

| Сценарий | Описание |
|----------|----------|
| ✋ Отказ | Мягко и ясно отклонять просьбы |
| ⏰ Напоминание | Продвигать дела, не обижая |
| 🙏 Извинение | Искренние извинения без самоуничижения |
| 💬 Недовольство | Выражать недовольство без ссор |
| 🚧 Границы | Чётко выражать свои ограничения |
| 👔 Разговор с руководителем | Профессиональное общение вверх |
| 🤝 Разговор с коллегами | Дружелюбное и деловое общение |
| 💕 Общение в паре | Чувства и потребности в отношениях |
| 🏠 Разговор с родителями | Объяснить трудные вещи родителям |
| 💰 Деньги | Неловкие разговоры о деньгах |
| 📞 Жалобы | Эффективные жалобы, дающие результат |
| 📋 Зарплата и увольнение | Общение в ключевые карьерные моменты |

## Быстрый старт

```bash
git clone https://github.com/xilin1214-jpg/saybetter.git
cd saybetter
pnpm install
cp .env.example .env
pnpm dev
```

## Конфиденциальность

- Ваши данные никогда не хранятся на наших серверах
- При самостоятельном хостинге все запросы идут через вашу конфигурацию
