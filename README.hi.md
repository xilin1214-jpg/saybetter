# SayBetter

> वही कहो जो कहना चाहते हैं, बिना किसी को ठेस पहुँचाए।

**SayBetter** एक ओपन-सोर्स कम्युनिकेशन असिस्टेंट है जो मुश्किल संदेशों को स्पष्ट, सौम्य और प्रभावी अभिव्यक्ति में बदलता है।

## उदाहरण

```
इनपुट: तुम फिर से लेट क्यों हो?

आउटपुट:
  ✅ सौम्य:    क्या हम इसे आज फाइनल कर सकते हैं? मुझे आगे की योजना के लिए इसकी ज़रूरत है।
  ✅ दृढ़:     यह आज फाइनल होना चाहिए, नहीं तो आगे का काम प्रभावित होगा।
               कृपया दिन के अंत तक स्पष्ट उत्तर दें।
  ✅ बुद्धिमान: मुझे पता है आपके पास बहुत कुछ है, लेकिन यह परिणाम आगे के लिए महत्वपूर्ण है।
               क्या हम आज निष्कर्ष पर पहुँच सकते हैं?
  ❌ न कहें:  "तुम हमेशा ऐसे हो" / "तुम कभी समय पर नहीं आते"
```

## SayBetter क्यों?

बहुत से लोगों के पास विचार नहीं होते — उनके पास शब्द नहीं होते। SayBetter आपको तब स्पष्ट रूप से व्यक्त करने में मदद करता है जब यह महत्वपूर्ण हो: मना करना, याद दिलाना, माफी मांगना, सीमाएँ设定 करना, आगे बढ़ाना।

## विशेषताएँ

- कठिन संदेशों को 3-5 बेहतर संस्करणों में बदलें
- रिश्ता, लक्ष्य और लहजा चुनें
- 12 परिदृश्य-आधारित प्रॉम्प्ट लाइब्रेरी (योगदान के लिए खुला)
- "अनुशंसित नहीं" अभिव्यक्तियाँ और व्याख्या
- साझा करने योग्य कार्ड (PNG डाउनलोड)
- 20 वास्तविक उदाहरण
- OpenAI-संगत (DeepSeek, Kimi, Ollama आदि के साथ काम करता है)
- 10 भाषाएँ समर्थित

| ✋ अस्वीकृति | अनुरोधों को सौम्यता से मना करें |
| ⏰ याद दिलाना | बिना अपमानजनक आगे बढ़ाएँ |
| 🙏 माफी | सच्ची माफी |
| 💬 असंतोष | बिना लड़ाई के असंतोष व्यक्त करें |
| 🚧 सीमाएँ | अपनी सीमाएँ स्पष्ट करें |
| 👔 प्रबंधक से बात | पेशेवर संवाद |
| 🤝 सहकर्मियों से बात | मित्रवत और कुशल संवाद |
| 💕 संबंध संवाद | रोमांटिक संबंधों में भावनाएँ |
| 🏠 माता-पिता से बात | कठिन बातें समझाएँ |
| 💰 पैसा | पैसे से जुड़ी असहज बातचीत |
| 📞 शिकायत | प्रभावी शिकायतें |
| 📋 वेतन और त्यागपत्र | महत्वपूर्ण करियर संवाद |

## त्वरित शुरुआत

```bash
cd saybetter
pnpm install
cp .env.example .env
pnpm dev
```
